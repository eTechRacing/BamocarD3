Firmware update-3  SD-Flash
============================

=> Please unpack "Firmware update inst.zip" and run the setup.
- You will need to configure the SD-Flash:
   - File -> SerialFlashreg2812.sdp   (~> MSG: Interface version 1)
   - Project -> Settings -> Target -> Emulator Address/Id: C1 (Configure your Com address here)
   - Project -> Settings -> Programming -> Flash Data File: (Select the Reg2812.out here)
   -> OK
   - Device -> Flash
